# Modern CSS Techniques

CSS has evolved significantly in recent years. Let's explore some modern techniques that can improve your stylesheets.

## CSS Grid vs Flexbox

Understanding when to use CSS Grid versus Flexbox is crucial for efficient layouts.

### CSS Grid
Perfect for two-dimensional layouts:

```css
.grid-container {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(300px, 1fr));
  gap: 1rem;
}
```

### Flexbox
Ideal for one-dimensional layouts:

```css
.flex-container {
  display: flex;
  justify-content: space-between;
  align-items: center;
}
```

## Custom Properties

CSS custom properties (variables) make themes and consistent design systems much easier to maintain.

```css
:root {
  --primary-color: #3b82f6;
  --secondary-color: #64748b;
  --border-radius: 0.5rem;
  --spacing-unit: 1rem;
}

.button {
  background-color: var(--primary-color);
  border-radius: var(--border-radius);
  padding: var(--spacing-unit);
}
```

## Container Queries

The future of responsive design is here with container queries, allowing components to respond to their container size.

```css
.card {
  container-type: inline-size;
}

@container (min-width: 300px) {
  .card-content {
    display: flex;
    flex-direction: row;
  }
}
```

## Logical Properties

Use logical properties for better internationalization support:

```css
/* Instead of margin-left */
margin-inline-start: 1rem;

/* Instead of padding-top */
padding-block-start: 1rem;
```

## Conclusion

These modern CSS techniques can significantly improve your development workflow and create more maintainable stylesheets.
