# RSSR v2 (SolidStart)

Stack:
- Bun
- SolidStart (minimal)
- Tailwind CSS
- TypeScript (strict)
- Biome

Scripts:
- bun run dev
- bun run build
- bun run preview
- bun run format
- bun run lint

Development:
- Pages only return Views (+ optional metadata)
- Views live in src/views and may default export
- Other modules use named exports only
