```python
# Python example with classes and functions
class Calculator:
    def __init__(self):
        self.history = []
    
    def add(self, a: int, b: int) -> int:
        result = a + b
        self.history.append(f"{a} + {b} = {result}")
        return result
    
    def get_history(self) -> list:
        return self.history

# Usage
calc = Calculator()
result = calc.add(5, 3)
```

## Testing Features

The enhanced CodeBlock should support:

1. **Language Detection**: Automatic detection from className or data-language
2. **GitHub Themes**: Both light and dark themes
3. **Copy Functionality**: Click to copy code to clipboard
4. **Line Numbers**: Optional line numbering
5. **Line Highlighting**: Highlight specific lines
6. **Accessibility**: Proper ARIA labels and keyboard navigation

## Copy Functionality Tests

Test the copy functionality with these methods:

1. **Click the Copy button** - Should show "Copied" feedback
2. **Keyboard navigation** - Tab to the copy button and press Enter/Space
3. **Ctrl+C shortcut** - Focus the code block and press Ctrl+C (or Cmd+C on Mac)
4. **Error handling** - Should show "Failed" if clipboard access is denied
